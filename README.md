# grpc-chat-sd
Simples projeto de um chat usando gRPC para a disciplina de Sistemas Distribuidos, IFPB.

Para rodar o projeto, execute os segintes comandos:
- npm install
- node server
- node client
- node client

Após executar o último comando, pelo menos duas vezes, e digitar o nome do usuário,
a conversa pode ser iniciada.
